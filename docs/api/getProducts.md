# Get Products

Give All the Products from the backend

### Endpoint

http://localhost:3000/api/getProducts

### Request

> Type: GET

### Parameters

```
Nan
```

### Example

```
curl -X GET http://localhost:3000/api/getProducts
```

### Response

Array of products from the backend

```json
[
  {
    "_id": "",
    "imgUrl": "",
    "productName": "",
    "price": 0,
    "__v": 0
  },
  ...
]
```
